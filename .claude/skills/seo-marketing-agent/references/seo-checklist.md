# SEO Checklist Expanded

Comprehensive SEO checklist for audits and optimization.

## On-Page SEO

### Title Tags

- [ ] Every page has a unique title
- [ ] Title length: 50-60 characters
- [ ] Primary keyword near the beginning
- [ ] Brand name at end (format: "Page Title | Brand")
- [ ] No keyword stuffing
- [ ] Compelling and click-worthy
- [ ] Matches search intent
- [ ] No duplicate titles across site

### Meta Descriptions

- [ ] Every page has a unique meta description
- [ ] Length: 150-160 characters
- [ ] Includes primary keyword naturally
- [ ] Contains a call-to-action
- [ ] Accurately describes page content
- [ ] Entices clicks from SERP
- [ ] No duplicate descriptions across site

### Heading Structure

- [ ] Single H1 per page
- [ ] H1 includes primary keyword
- [ ] H1 matches/relates to title tag
- [ ] Logical hierarchy: H1 → H2 → H3 → H4
- [ ] No skipped heading levels
- [ ] Headings describe section content
- [ ] Keywords in subheadings where natural
- [ ] Headings aid scanability

### URL Structure

- [ ] Descriptive URLs (not /page?id=123)
- [ ] Includes target keyword
- [ ] All lowercase
- [ ] Words separated by hyphens
- [ ] Short and simple (< 75 characters)
- [ ] No unnecessary parameters
- [ ] No session IDs in URLs
- [ ] Consistent URL structure across site

### Content Optimization

- [ ] Primary keyword in first 100 words
- [ ] Natural keyword density (1-2%)
- [ ] Related keywords and synonyms used
- [ ] Sufficient content depth (> 300 words for most pages)
- [ ] Unique content (not duplicated)
- [ ] Content matches search intent
- [ ] Updated/fresh content where relevant
- [ ] No thin content pages

### Internal Linking

- [ ] Logical site structure
- [ ] Important pages within 3 clicks of home
- [ ] Descriptive anchor text
- [ ] Related pages linked together
- [ ] No orphan pages
- [ ] Reasonable number of links per page
- [ ] Broken internal links fixed

### Images

- [ ] Descriptive file names (not IMG_1234.jpg)
- [ ] Alt text on all images
- [ ] Alt text is descriptive and includes keywords where relevant
- [ ] Images compressed and optimized
- [ ] Appropriate file format (WebP, JPEG, PNG)
- [ ] Responsive images (srcset)
- [ ] Lazy loading for below-fold images
- [ ] Width and height attributes set

## Technical SEO

### Crawlability

- [ ] Robots.txt exists and is correct
- [ ] Important pages not blocked
- [ ] XML sitemap exists
- [ ] Sitemap submitted to Search Console
- [ ] Sitemap updated automatically
- [ ] No crawl errors in Search Console
- [ ] Reasonable crawl budget usage

### Indexability

- [ ] Important pages return 200 status
- [ ] No accidental noindex tags
- [ ] Canonical tags set correctly
- [ ] No duplicate content issues
- [ ] Pagination handled correctly (rel=prev/next or alternatives)
- [ ] Parameter handling configured
- [ ] Hreflang tags for multilingual sites

### Site Architecture

- [ ] Flat architecture (important pages near root)
- [ ] Logical category structure
- [ ] Breadcrumb navigation
- [ ] HTML sitemap for users
- [ ] Faceted navigation handled correctly
- [ ] URL parameters under control

### HTTPS & Security

- [ ] SSL certificate installed
- [ ] All pages served over HTTPS
- [ ] HTTP redirects to HTTPS
- [ ] No mixed content warnings
- [ ] HSTS header configured
- [ ] Certificate not expired

### Mobile

- [ ] Mobile-responsive design
- [ ] Viewport meta tag present
- [ ] Touch targets ≥ 48px
- [ ] No horizontal scrolling
- [ ] Text readable without zooming (≥ 16px)
- [ ] Mobile-friendly test passes
- [ ] No intrusive interstitials

### Page Speed

- [ ] LCP < 2.5 seconds
- [ ] FID/INP < 100/200ms
- [ ] CLS < 0.1
- [ ] Time to First Byte < 600ms
- [ ] Total page size reasonable (< 3MB)
- [ ] Number of requests minimized
- [ ] Critical rendering path optimized

### Structured Data

- [ ] Schema.org markup present
- [ ] Correct schema type for content
- [ ] Validated with testing tool
- [ ] No errors or warnings
- [ ] Rich results appearing (if applicable)

## Local SEO (if applicable)

- [ ] Google Business Profile claimed
- [ ] NAP consistent across web
- [ ] Local keywords in content
- [ ] Local structured data
- [ ] Reviews encouraged and managed
- [ ] Local citations built

## International SEO (if applicable)

- [ ] Hreflang tags implemented correctly
- [ ] Language targeting configured
- [ ] Country-specific domains/subdirectories
- [ ] Content properly localized
- [ ] Search Console international targeting set

## Off-Page SEO

### Link Profile

- [ ] Quality backlinks being acquired
- [ ] No toxic/spammy links
- [ ] Diverse anchor text profile
- [ ] Links from relevant sites
- [ ] Disavow file if needed

### Social Signals

- [ ] Social profiles active
- [ ] Content shareable
- [ ] Open Graph tags present
- [ ] Twitter Card tags present
- [ ] Social sharing buttons (where appropriate)

## Analytics & Tracking

- [ ] Google Analytics installed
- [ ] Google Search Console verified
- [ ] Bing Webmaster Tools verified
- [ ] Goals/conversions configured
- [ ] Event tracking for key actions
- [ ] Site search tracking enabled
- [ ] Filters for internal traffic
- [ ] Regular reporting scheduled

## Content Audit

### Quality Assessment

- [ ] No duplicate content
- [ ] No thin content (< 200 words with no purpose)
- [ ] No outdated information
- [ ] No broken external links
- [ ] Content provides value
- [ ] Content matches user intent
- [ ] E-E-A-T signals present (expertise, experience, authority, trust)

### Content Gaps

- [ ] Competitor content analyzed
- [ ] Keyword gaps identified
- [ ] Content calendar planned
- [ ] Evergreen content strategy
- [ ] Content refresh schedule

## Common Issues Checklist

### Critical Issues

- [ ] No noindex on important pages
- [ ] No robots.txt blocking
- [ ] No manual penalties
- [ ] No severe Core Web Vitals issues
- [ ] HTTPS working correctly
- [ ] No major duplicate content

### High Priority

- [ ] Missing/poor title tags
- [ ] Missing meta descriptions
- [ ] Poor mobile experience
- [ ] Slow page speed
- [ ] Missing alt text
- [ ] Broken links

### Medium Priority

- [ ] Missing structured data
- [ ] Suboptimal URL structure
- [ ] Thin content pages
- [ ] Missing internal links
- [ ] Keyword cannibalization

### Low Priority

- [ ] Minor meta description length issues
- [ ] Images could be more optimized
- [ ] Additional schema opportunities
- [ ] Content freshness updates

## Monthly SEO Tasks

- [ ] Review Search Console for issues
- [ ] Check Core Web Vitals
- [ ] Review top performing pages
- [ ] Review declining pages
- [ ] Check for new crawl errors
- [ ] Review backlink profile
- [ ] Update/refresh key content
- [ ] Check competitor changes
